# v1.4.3 — 文档重构与常用配置集

## 背景

随着功能迭代，README 内容逐渐膨胀，结构不够清晰，缺少目录导航，示例展示方式（纯代码块）不够直观。同时用户反馈缺少开箱即用的配置参考。

## 需求

1. 重构中英文 README，优化文档结构和可读性
2. 提供常用配置集参考文档，降低用户上手门槛
3. 更新演示 GIF，展示最新功能

## 方案

### README 重构

- 新增目录导航（Table of Contents），方便快速定位
- 用 HTML 表格替代纯代码块展示示例，同时呈现配置和效果
- 精简格式函数表，去除冗余列和重复示例
- 重组章节结构：配置 → 模板语法 → 高级功能 → 自定义函数 → 调试
- 中英文 README 保持一致的新结构

### 常用配置集文档

- 新增 `doc/rules/cn.md`（中文）和 `doc/rules/en.md`（英文）
- 提供分类整理的常用规则配置，用户可直接复制使用
- 在 README 中添加指向配置集文档的链接

### 其他

- 更新 `public/start.gif` 演示动图
- 删除临时文件 `doc/rules/cn/temp.md`
